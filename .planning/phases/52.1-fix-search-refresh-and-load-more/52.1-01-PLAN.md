Structure of unified do_search():

1. **Signature:** `do_search <- function(mode = c("refresh", "load_more"))`
   - `mode <- match.arg(mode)` at top

2. **Common preamble:**
   - `nb_id <- notebook_id(); req(nb_id)`
   - `nb <- get_notebook(con(), nb_id); req(nb$type == "search")`

3. **Mode-specific guards (BEFORE common logic):**
   - refresh: `req(!is.null(nb$search_query) && nchar(nb$search_query) > 0)`
   - load_more: Replace bare `req(pagination_state$has_more)` with `if (!isTRUE(pagination_state$has_more)) { showNotification("No more results available", type = "message"); return() }`. Then `req(!is_processing()); is_processing(TRUE)`. Then spinner swap + on.exit() for guaranteed restore.

4. **Common config and cursor setup:**
   - `cfg <- config(); newly_added <- 0L`
   - `cursor <- if (mode == "refresh") NULL else pagination_state$cursor`
   - `progress_msg <- if (mode == "refresh") "Searching OpenAlex..." else "Loading more papers..."`

5. **API call with FIXED error handling (critical bug fix):**
   - Wrap the entire withProgress block in tryCatch
   - Error handler: classify error, show_error_toast, then `return()` â€” this return() exits do_search() entirely, preventing ANY downstream toast
   - Do NOT return `list(papers = list())` from error handler â€” that was the triple-toast bug
   - Inside withProgress: email/api_key retrieval, filter parsing (with NULL safety), search_papers() call with cursor, pagination_state update, paper processing

6. **Paper processing (inside withProgress, after API call):**
   - Update pagination_state (cursor, has_more, api_total)
   - Check `length(papers) == 0` â€” show appropriate toast and return()
   - Filter excluded papers (existing excluded_ids logic)
   - DB dedup loop (existing pattern: check existing by paper_id, skip if exists, create_abstract + create_chunk)
   - Use `newly_added <<- newly_added + 1L` (note <<- for assignment inside for loop within withProgress)

7. **Post-withProgress (only reached on success):**
   - `paper_refresh(paper_refresh() + 1)`
   - `total_in_nb <- nrow(list_abstracts(con(), nb_id))`
   - `pagination_state$total_fetched <- total_in_nb`
   - Mode-specific success toast:
     - refresh + 0 added: "No new papers found"
     - refresh + N added: "Added N new paper(s) (X total in notebook)"
     - load_more + N added: "Loaded N more paper(s) (X total)"
     - load_more + 0 added: "No new papers (all duplicates)"

8. **Update observeEvent handlers:**
   - `observeEvent(input$refresh_search, { do_search("refresh") }, ignoreInit = TRUE, ignoreNULL = TRUE)`
   - `observeEvent(input$load_more, { do_search("load_more") }, ignoreInit = TRUE, ignoreNULL = TRUE)`

9. **NULL safety audit â€” check ALL if() conditions in the function:**

   - `filters$search_field %||% "default"` â€” safe
   - `filters$is_oa %||% FALSE` â€” safe
