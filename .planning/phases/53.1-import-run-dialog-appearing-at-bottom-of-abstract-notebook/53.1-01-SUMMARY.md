# Phase 53.1 Plan 01: Import Run History UI Fix

Fix import run history rendering bug and restructure history as optional modal footer feature.

## One-Liner

Parameterized bulk import module to optionally show collapsible history in modal footer, eliminating page-level rendering bug where history appeared at bottom of abstract notebook.

## What Was Built

### Core Changes

1. **Module Parameterization (Task 1)**
   - Added `show_history = FALSE` parameter to `mod_bulk_import_ui()` and `mod_bulk_import_server()`

   - Both `show_history = TRUE` and `show_history = FALSE` output only JS script tag at module level
   - History now renders inline in `show_import_modal()` closure as part of modal footer

   - When `show_history = TRUE`, modal footer includes collapsible history section below action buttons

   - Empty state: "No imports yet" message when no runs exist
   - Delete buttons completely removed from history cards
   - All element IDs properly namespaced with `ns()`

## How It Works

### Before (Bug State)

```r
mod_bulk_import_ui <- function(id) {
  tagList(
    tags$script(src = "js/import-progress.js"),
    uiOutput(ns("import_history"))  # Rendered at module insertion point!
  )
}
```

When `mod_bulk_import_ui(ns("bulk_import"))` was called in `mod_search_notebook.R`, the `uiOutput(ns("import_history"))` rendered directly into the page flow at the bottom of the abstract notebook sidebar, causing unstyled history cards to appear below all other content.

### After (Fixed State)

```r
mod_bulk_import_ui <- function(id, show_history = FALSE) {
  tagList(
    tags$script(src = "js/import-progress.js")
    # No uiOutput - history moved into modal
  )
}
```

History now renders inline in `show_import_modal()` as part of the modal dialog's footer content when `show_history = TRUE`. The modal is created on-demand, so history only appears inside the modal dialog, never in page-level flow.

### History Section Structure

```r
if (show_history) {
  # Fetch runs
  runs <- get_import_runs(con(), notebook_id())

  # Build collapsible section
  history_section <- tags$div(
    style = "width: 100%; margin-top: 1rem;",
    tags$hr(),
    # Toggle header with count badge
    tags$div(
      `data-bs-toggle` = "collapse",
      `data-bs-target` = paste0("#", ns("import_history_collapse")),
      tags$span("Past Imports ", tags$span(class = "badge", nrow(runs)))
    ),
    # Collapse target (collapsed by default)
    tags$div(
      id = ns("import_history_collapse"),
      class = "collapse",
      # Last 5 run cards or empty state
    )
  )

  # Append to footer
  footer_content <- c(footer_content, list(history_section))
}
```

### Caller Pattern

**Search Notebook (wants history):**
```r
mod_bulk_import_ui(ns("bulk_import"), show_history = TRUE)
bulk_import_api <- mod_bulk_import_server("bulk_import", ..., show_history = TRUE)
```

**Sidebar Import (no history):**
```r
mod_bulk_import_ui("sidebar_import")  # Default show_history = FALSE
sidebar_import_api <- mod_bulk_import_server("sidebar_import", ...)  # Default
```
