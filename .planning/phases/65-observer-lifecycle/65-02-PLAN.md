**mod_document_notebook.R — Add session$onSessionEnded cleanup:**

1. Before the closing `})` of the moduleServer callback (near end of file, before `}` that closes the moduleServer function), add:

```r
    # LIFE-04: Clean up observer handles on session end to prevent orphaned references.
    # Follows mod_citation_network.R cleanup pattern.
    session$onSessionEnded(function() {
      # Destroy figure action observers
      for (id in names(fig_action_observers)) {
        obs_list <- fig_action_observers[[id]]
        if (is.list(obs_list)) {
          for (obs in obs_list) {
            if (!is.null(obs)) tryCatch(obs$destroy(), error = function(e) NULL)
          }
        }
        fig_action_observers[[id]] <- NULL
      }
      # Destroy extract button observers
      for (id in names(extract_observers)) {
        obs <- extract_observers[[id]]
        if (!is.null(obs)) tryCatch(obs$destroy(), error = function(e) NULL)
        extract_observers[[id]] <- NULL
      }
      # Destroy delete document observers
      for (id in names(delete_doc_observers)) {
        obs <- delete_doc_observers[[id]]
        if (!is.null(obs)) tryCatch(obs$destroy(), error = function(e) NULL)
        delete_doc_observers[[id]] <- NULL
      }
    })
```

Key points:
- Use `tryCatch(obs$destroy(), error = function(e) NULL)` for each destroy call (handles already-destroyed observers)
- Clean up ALL three observer stores: `fig_action_observers`, `extract_observers`, `delete_doc_observers`
- Assign NULL after destroy to clear the reference from the reactiveValues store
- Place this INSIDE the moduleServer callback, just before the closing braces

2. Find the exact insertion point: Look for the last `observeEvent` or `output$` assignment in the moduleServer body. The cleanup hook goes after all reactive/observer registrations but before the closing `})` of the moduleServer.
