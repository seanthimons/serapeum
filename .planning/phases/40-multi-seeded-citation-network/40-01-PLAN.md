  <name>Task 1: Create multi-seed BFS function and update build_network_data shapes</name>
  <files>R/citation_network.R</files>
  <action>
  1. Add `fetch_multi_seed_citation_network()` function after the existing `fetch_citation_network()`:
     - Signature: `fetch_multi_seed_citation_network(seed_paper_ids, email, api_key = NULL, direction = "both", depth = 2, node_limit_per_seed = 100, interrupt_flag = NULL, progress_file = NULL)`
     - `seed_paper_ids` is a character vector of OpenAlex Work IDs
     - Loop over each seed, calling existing `fetch_citation_network()` per seed with `node_limit = node_limit_per_seed`
     - Before each seed BFS, write seed-level progress: `write_progress(progress_file, seed_idx, total_seeds, 0, 1, sprintf("Processing seed %d of %d...", seed_idx, total_seeds))`
     - Check interrupt_flag between seeds (early exit if interrupted)
     - After all seeds: rbind all node data frames, rbind all edge data frames
     - Deduplicate nodes by `paper_id` (keep first occurrence)
     - Deduplicate edges by `from_paper_id` + `to_paper_id` pair
     - Re-mark seeds: `merged_nodes$is_seed <- merged_nodes$paper_id %in% seed_paper_ids` (prevents lost seed markers per Pitfall 3 in research)
     - Compute overlap: count how many per-seed result sets contain each paper_id. Set `merged_nodes$is_overlap <- overlap_count >= 2 & !merged_nodes$is_seed` (seeds are NOT overlap even if reachable from multiple seeds)
     - Return `list(nodes = merged_nodes, edges = merged_edges, partial = any_partial)`
     - Handle edge case: if `length(seed_paper_ids) == 1`, delegate directly to `fetch_citation_network()` and set `is_overlap = FALSE` on all nodes (backward compatibility)

  2. Update `build_network_data()`:
     - Change shape assignment from `ifelse(nodes_df$is_seed, "star", "dot")` to:
       ```r
       nodes_df$shape <- ifelse(
         nodes_df$is_seed,
         "star",
         ifelse(
           isTRUE(nodes_df$is_overlap) | (!is.null(nodes_df$is_overlap) & nodes_df$is_overlap),
           "diamond",
           "dot"
         )
       )
       ```
     - Handle the case where `is_overlap` column doesn't exist (loaded from old saved networks): default to `FALSE`
     - Update `seed_paper_id` parameter to accept either single ID or vector. When vector, mark all as seeds: `nodes_df$is_seed <- nodes_df$paper_id %in% seed_paper_id`

  AVOID: Don't modify the existing `fetch_citation_network()` signature — it remains the single-seed workhorse called by the multi-seed wrapper.
