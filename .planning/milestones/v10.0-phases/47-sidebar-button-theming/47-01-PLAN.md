  **Step 2: Create automated verification test.**

  Create `tests/testthat/test_icon_wrappers.R` that:
  1. Greps all R files (R/*.R and app.R) for raw `icon("` calls
  2. Excludes lines inside wrapper DEFINITIONS (in theme_catppuccin.R where `shiny::icon(` is the implementation)
  3. Asserts zero raw icon() calls remain outside wrapper definitions
  4. Tests that all wrapper functions exist and are callable

  ```r
  test_that("no raw icon() calls remain outside wrapper definitions", {
    r_files <- c(list.files("R", pattern = "\\.R$", full.names = TRUE), "app.R")
    raw_calls <- character(0)
    for (f in r_files) {
      lines <- readLines(f)
      # Find lines with icon(" but NOT shiny::icon(" (which are wrapper definitions)
      matches <- grep('(?<!shiny::)icon\\("', lines, perl = TRUE)
      # Also exclude icon( used inside wrapper function bodies in theme_catppuccin.R
      if (basename(f) == "theme_catppuccin.R") {
        # Wrapper definitions use shiny::icon, so the negative lookbehind handles them
        next
      }
      if (length(matches) > 0) {
        raw_calls <- c(raw_calls, paste0(f, ":", matches, ": ", trimws(lines[matches])))
      }
    }
    expect_equal(length(raw_calls), 0,
      info = paste("Raw icon() calls found:\n", paste(raw_calls, collapse = "\n")))
  })
  ```
