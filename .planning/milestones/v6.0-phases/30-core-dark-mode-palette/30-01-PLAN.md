3. **catppuccin_dark_css() function** — Returns a single CSS string with all `[data-bs-theme="dark"]` overrides:
   - Body colors: --bs-body-bg (Mocha base), --bs-body-color (Mocha text)
   - Secondary/tertiary bg: --bs-secondary-bg (surface0), --bs-tertiary-bg (surface1)
   - Primary/link: --bs-primary (lavender), --bs-link-color (sapphire), --bs-link-hover-color (sky)
   - Semantic: --bs-success (green), --bs-danger (red), --bs-warning (yellow), --bs-info (blue)
   - Borders: --bs-border-color (surface2)
   - Card bg: --bs-card-bg (surface0), --bs-card-border-color (surface1)
   - Input bg: --bs-body-bg for inputs, border using surface2
   - Semantic RGB variants for Bootstrap's rgba() usage: --bs-primary-rgb, --bs-success-rgb, etc. (convert hex to R,G,B)
   - Toast overrides: .shiny-notification-message (green bg + base text), .shiny-notification-warning (yellow bg + base text), .shiny-notification-error (red bg + base text)

Include a helper `hex_to_rgb()` that converts "#1e1e2e" to "30,30,46" for Bootstrap RGB variable needs.

**app.R changes:**

   ```r

     version = 5,
     preset = "shiny",
     bg = LATTE$base,
     fg = LATTE$text,
     primary = LATTE$lavender,
     secondary = LATTE$surface1,
     success = LATTE$green,
     danger = LATTE$red,
     warning = LATTE$yellow,
     info = LATTE$blue,
   ```

2. After the bs_theme() call, add bs_add_rules() to inject the dark mode CSS. Use a pattern like:
   ```r

   ```

   ```r
   serapeum_theme <- bs_theme(...)
   serapeum_theme <- bs_add_rules(serapeum_theme, catppuccin_dark_css())

   ui <- page_sidebar(
     ...
     theme = serapeum_theme,
     ...
   )
   ```

3. Update the dark mode toggle localStorage restore script. The existing toggle button in the title bar works correctly — keep it as-is. But ensure the initial page load restores the saved theme. If there's already a script for this, keep it. If not, add one in tags$head:
   ```js
   // Restore saved theme on page load
   (function() {
     const saved = localStorage.getItem('theme');
     if (saved) document.documentElement.setAttribute('data-bs-theme', saved);
   })();
   ```
