Update www/custom.css to fully adapt the citation network for dark mode using Catppuccin Mocha colors:

1. **Canvas background** — Add rule to force vis.js canvas dark in dark mode:
```css
[data-bs-theme="dark"] .citation-network-container .vis-network canvas {
  background-color: #1e1e2e !important;
}
```
The `!important` is necessary here because vis.js sets canvas background programmatically. This is one of the few justified uses of !important.

2. **Legend panel** — Update existing dark mode legend rules (lines 109-124) to use Catppuccin Mocha values:
```css
[data-bs-theme="dark"] .citation-network-legend {
  background-color: rgba(49, 50, 68, 0.95);  /* Mocha Surface0 at 95% */

  box-shadow: 0 2px 6px rgba(0, 0, 0, 0.3);  /* Darker shadow for dark mode */
}

[data-bs-theme="dark"] .citation-network-legend h6 {
  color: #cdd6f4;  /* Mocha Text */
}

[data-bs-theme="dark"] .citation-network-legend .legend-toggle {
```
