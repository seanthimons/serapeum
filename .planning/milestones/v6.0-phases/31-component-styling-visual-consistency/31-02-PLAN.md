**Spacing audit (UIPX-01):**

Scan all module UI functions for spacing inconsistencies. The app should follow Bootstrap's spacing scale:
- Small gaps: `gap-1`, `py-1`, `px-1` (0.25rem)
- Standard gaps: `gap-2`, `py-2`, `px-2` (0.5rem)
- Section gaps: `mb-3`, `mt-3`, `py-3` (1rem)
- Large gaps: `mb-4`, `py-4` (1.5rem)
- Extra large: `py-5` (3rem)

Check for any hardcoded pixel values in style= attributes that should use Bootstrap spacing utilities instead. Fix any inconsistencies found.

**Typography audit (UIPX-02):**

Verify heading hierarchy is consistent across modules:
- Module headers: h4 (e.g., notebook names, section titles)
- Sub-sections: h5 (e.g., "Core Technologies", "Key Packages")
- Small labels: h6 or `.small.fw-semibold` (e.g., sidebar section labels)
- Body text: Default Bootstrap (1rem, line-height 1.5)

Fix any inconsistencies where heading levels skip or fonts/weights mismatch.
