## Key Insight

The autonomous research agent must not make research easier. It must make research **harder — at a higher level**. It is a problem generator, not an answer generator. Its primary output is contradictions, gaps, and cross-disciplinary tensions invisible at human traversal scale.

## Key Metaphors

- **Arrakis vs. Caladan** — the tool must create a demanding environment, not a comfortable one
- **Shai-Hulud** — the agent is simultaneously the danger and the spice
- **Persistence Hunting** — the agent is endurance, the human is the tracker's eye
- **The Farsi Proverb** — "woe to the day the salt itself has spoiled"
- **The Dune Quote** — restored as design specification, not warning
